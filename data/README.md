# ShopSense

Semantic Product Retrieval System built using Amazon ESCI Dataset, Sentence Transformers, FAISS, and Streamlit.

## Problem

Traditional keyword search often fails to understand user intent.

Example:

Query:
gift for mom

Desired Result:
Relevant gifts for mothers even when exact keywords are not present.

## Solution

ShopSense uses dense vector embeddings generated by Sentence Transformers and performs semantic retrieval using FAISS.

## Dataset

Amazon ESCI Dataset

- 20,000 Products
- 52,687 Query-Product Pairs
- 13,712 Unique Queries

## Architecture

User Query
↓
Sentence Transformer
↓
384-D Embedding
↓
FAISS Similarity Search
↓
Top-10 Ranked Results

## Evaluation

| Metric | Score |
|----------|----------|
| Recall@10 | 0.6360 |
| MRR@10 | 0.3847 |
| NDCG@10 | 0.3710 |
