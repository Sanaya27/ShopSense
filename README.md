# ShopSense

Intent-Aware Product Search Engine

ShopSense is a semantic product search system that understands user intent rather than relying solely on keyword matching.

## Features

- Semantic Search
- Product Retrieval
- FAISS Vector Search
- Streamlit Interface
- Amazon ESCI Dataset

## Tech Stack

- Python
- Sentence Transformers
- FAISS
- Streamlit
- GitHub
